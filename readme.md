## 💡 Sobre o Projeto
**Boletogen** é um sistema de geração de boletos bancários em Java, compatível com:
- ✅ **Banco do Brasil**
- ✅ **Bradesco**  
- ✅ **Itaú**  

Gera PDFs prontos para impressão com:  
- Código de barras válido  
- Linha digitável formatada  
- Dados do beneficiário/pagador
